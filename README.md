## Краткое описание проекта about:Credit

**Назначение:** Android-приложение для расчёта и сравнения условий финансовых продуктов (кредит, микрозайм, ипотека).

**Функционал:**
- Расчёт ежемесячного и ежедневного платежа, переплаты, эффективной ставки
- Настройка параметров (ставка, срок, комиссии) для каждого типа продукта
- Сохранение истории расчётов в локальное хранилище
- Просмотр и очистка истории

**Архитектура:**
- **CalcActivity** — ввод данных, запуск расчёта, сохранение результата
- **CreditCalculator** — бизнес-логика, финансовые формулы
- **JournalActivity** — отображение сохранённых расчётов
- **SharedPreferences** — единое хранилище настроек и истории (JSON)

**Особенности:**
- Все данные хранятся локально, без сервера
- Микрозаймы поддерживают расчёт по дням и месяцам
- Настройки изменяются через долгое нажатие на кнопки
- Минимальные зависимости, чистая реализация на Java
